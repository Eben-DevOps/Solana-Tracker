# Crypto Wallet Tracker
This project tracks wallet activities on the Solana blockchain, logs transactions to an SQLite database, and sends alerts via Telegram.

## Setup

1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Configure your settings in `config/settings.py`.
4. Run the tracker:
   ```bash
   python main.py
   ```
